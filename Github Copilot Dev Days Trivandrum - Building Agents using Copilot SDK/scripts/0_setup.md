# Setup

- Install Github Copilot CLI locally. Setup instructions: https://docs.github.com/en/copilot/how-tos/copilot-cli/set-up-copilot-cli/install-copilot-cli
- Login using `copilot login` command.
- Open copilot CLI using `copilot` command on the working directory.
- Various `/` commands are available in Copilot CLI to connect tools, switch model, etc.
- For local models, install Ollama / LM Studio (https://lmstudio.ai/download)
